# PoYo Python Examples

These examples use `requests` and read `.env` from the repo root.

| Example | What it shows |
| --- | --- |
| [`gpt-image-2`](gpt-image-2/) | Submit image task, store `task_id`, poll status. |
| [`seedance-2`](seedance-2/) | Submit video task, store `task_id`, poll status. |
| [`gpt-5.2`](gpt-5.2/) | Send a synchronous chat completion request. |

Install dependencies per example:

```bash
pip install -r requirements.txt
```
