## Post 2 - Developer Education

```text
Most media generation APIs should not be called from browser code.

With PoYo, keep the API key server-side:
1. your app route accepts the prompt
2. server calls PoYo
3. client stores task_id
4. app polls status or receives a webhook

https://docs.poyo.ai/api-manual/task-management/status
```
